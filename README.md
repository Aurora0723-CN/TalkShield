# TalkShield

**TalkShield: Explainable Detection of Generative Talking-Head Forgeries via Multimodal Evidence Fusion**

**TalkShield：面向生成式 Talking-Head 伪造的可解释检测与多模态证据融合**

## Overview

This repository will host the official implementation of TalkShield, a research project for detecting generative talking-head forgeries and providing evidence-oriented explanations for model decisions.

Recent talking-head generation models can synthesize highly realistic talking-face videos, making reliable forgery detection increasingly challenging. TalkShield studies how visual and audio evidence can be combined to support video-level detection while keeping the decision process traceable and interpretable.

The project focuses on:

- detecting generative talking-head forgeries at the video level;
- organizing multimodal evidence from video and audio signals;
- highlighting suspicious temporal segments and facial regions;
- producing concise evidence summaries for detection results.

## 项目简介

本仓库将用于发布 TalkShield 的官方实现。TalkShield 面向生成式 talking-head 伪造检测任务，关注如何在视频级真伪判别中结合多模态证据，并为模型决策提供可追溯、可解释的依据。

随着 talking-head 生成模型能够合成越来越逼真的说话人视频，可靠的伪造检测变得更加困难。TalkShield 研究如何联合利用视觉与音频线索，在完成视频级检测的同时，定位关键时间片段和可疑面部区域，从而提升检测结果的可信度与可解释性。

本项目主要关注：

- 视频级生成式 talking-head 伪造检测；
- 视频与音频信号中的多模态证据组织；
- 可疑时间片段与面部区域定位；
- 面向检测结果的简洁证据摘要生成。
